# Execution Plans

Use this directory for versioned plans that should survive beyond one chat
session.

- `active/`: plans currently being executed.
- `completed/`: plans kept for decision history.
- [tech-debt-tracker.md](tech-debt-tracker.md): follow-up cleanup and quality
  gaps.

See [../PLANS.md](../PLANS.md) for when to create a plan and the preferred
template.
